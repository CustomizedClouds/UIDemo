UI Demos

Click On A Demo To Get Started

Data Tables

Data tables are a critical component of any application. Once connected to your server they will be populated with data from your app. Click in the search box on the top right to filter the results.

Forms

Handling form data is what it's all about. You can mix and match any of the components in the demo to build complex forms

Calendar

This calendar is built on a powerful framework called Full Calendar. It intergates perfectly with Google Calendar and a number of other calendar plugins as well. Toggle between the calendar views at the top right to see the different layouts.

Inbox

This inbox view is a Gmail-style email inbox and you can not only pull in all your emails from Gmail but also from any RSS or ATOM feed as well

Timeline

This timeline template is great for pulling chronilogical events from a database, similar to a Facebook feed. It's also a great way of constructing a visual representation of logging events.

Users

This users view is a nice and simple template you can employ in your application to track user events in real time. Combine it with the notifications template and you can receive status events in real-time!

Compose

This compose template is an email style compose form that can be connected to Gmail or any other email server.

Charts

These are a few different charts you can use to get visual representations of your data. They're great to use inside dashboard views for quick snapshots of what's going on system wide.

Settings

Adjusting system settings allows for a more dynamic application. Either one of these demos can be employed to give users a more fine-tuned experience when using your app. Our favorite is the knob settings!

Notifications

Notifications are a must for any real-time application. We can hook into any part of your application and fire alerts to keep you on top of things!

File Gallery

With this file gallery widget you can browse files on your server. You can even give everyone of your users their own file folders.

File Uploader

This is widget you can use to upload files to your server. Drag an image in the upload zone to see how it works.

Login Page

If you're going to give your users account space then locking down your application with a login page is a must. We can also create social login widgets to login with your social network of choice.

Todo Lists

This todo app is another widget we can connect to Gmail. Or we can have it pull from your database, you make the call!

Presence

With this widget we can create an online indicator that can be syndicated across all of your current logged in users. We can attach it to a chat server so you can see who's online.

Form Wizards

Form wizards are really useful when you want your users to do something in a succession. Fill out the form to see how the wizard works

UI Demo ReadMe

We created this app to showcase some of the more common UI components that we use when building applications. Whenever you launch a demo, try and picture what you're seeing populated with your data. To find out more about how this application works see below and to launch the UI demos click the 'Launch Demos' button on the home screen.


More About The CloudClient

This UI Demo is wrapped in an application we created called the Cloud Client that comprises a number of components for building desktop applications (mobile version coming soon!). Read on to find out how it works.

Runtime

The runtime for the Cloud Client comprises two seperate windows that talk to each other. Whenever you opened this app the runtime started and minimized itself to your system tray (it looks like the icon to the left). You can launch this app by clicking on the icon. Close the entire app by clicking on the 'Close App' button on the home screen.

Keycuts

If you're not using keyboard shortcuts you should be. They are real time savers and the Cloud Client comes embedded with a number of default shortcuts. You can add any shortcuts you want when building your app. (ex: hold down the ctrl key and press c to launch the console window for the app or ctrl + r to restart it)

Total Control

The Cloud Client can control applications on your computer in various ways. This can lead to creating dynamic interactions inside your app. For instance lets say you wanted to create a keyboard shortcut that opened your default browser and accessed Google when you pressed ctrl + g, no problem (try it). We have to admit that one was pre-programmed, the Cloud Client can't really read minds...YET!

FileIO

The term IO is geek-speak that stands for inputs and outputs. Basically the term is short form for reading and writing files similar to creating a new file inside notepad or other text editor. The Cloud Client can read and write files to your computer. For example click here to open the folder the app came in, here to create a new folder inside that folder and here to create a new file inside that new folder (say that five times fast!). Then open your new folder to see your new file.

Embedded Server

The Cloud Client comes with a high powered embedded server on-board. For instance whenever you launched the app it actually started a server on your computer. Don't believe me, click here to check it out in your own browser! How you use these servers is a bit more complex and will depend on the use case involved but if you need it, it's raring to go!
